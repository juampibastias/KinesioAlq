Imágenes y Descripción

Consultorio Privado

Este consultorio cuenta con escritorio con cajones, sillas para profesional y paciente, camilla, balanza, aire frío/calor, modular.

Box y gimnasio

Nuestro centro cuenta con dos modernos box completamente equipados para brindarle el mejor servicio. Cada box está provisto de 2 camillas cómodas, percheros para su comodidad, un sistema de aire acondicionado con regulación de temperatura y una iluminación ajustable para crear un ambiente ideal. Además, ponemos a su disposición equipos de rehabilitación Meditea de última generación, asegurando un enfoque terapéutico de vanguardia. Además de nuestros box, también contamos con un amplio gimnasio diseñado especialmente para la rehabilitación terapéutica. Este espacio cuenta con las dimensiones adecuadas para realizar ejercicios terapéuticos de manera cómoda y efectiva. Los espejos instalados en el gimnasio permiten una supervisión precisa de los movimientos, optimizando el proceso de recuperación. Al igual que en los box, el gimnasio está equipado con aire acondicionado, proporcionando un ambiente confortable durante las sesiones de entrenamiento. En nuestro gimnasio también ofrecemos la posibilidad de clases grupales, brindando flexibilidad y adaptabilidad para satisfacer las necesidades de nuestros profesionales y pacientes. Si se requiere un espacio completamente libre para las clases grupales, podemos acomodar esta solicitud de manera eficiente.

Sala de espera

Nuestra sala de espera está diseñada para brindarles una experiencia cómoda y segura mientras esperan. Disfruten del ambiente climatizado y la conveniencia de nuestro dispensador de agua fría y caliente. Hemos implementado una puerta de apertura interna única para aumentar la seguridad y privacidad de nuestros pacientes. La sala de espera puede acomodar hasta 4 pacientes, asegurando tiempos de espera mínimos y una atención más personalizada.
